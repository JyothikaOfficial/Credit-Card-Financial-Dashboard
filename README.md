💳 Credit Card Financial Dashboard

📌 Project Overview

The Credit Card Financial Dashboard is an interactive Power BI project developed to analyze credit card transaction performance, customer spending behavior, and revenue trends. The dashboard provides valuable insights into customer demographics, spending categories, geographic transaction distribution, and overall financial performance.

This project demonstrates how data visualization and business intelligence can support data-driven decision-making in the financial services domain.

🎯 Business Objective
The primary objectives of this project are:

Analyze customer spending behavior across various categories.
Monitor revenue generated from credit card transactions.
Identify high-value customer segments.
Evaluate transaction trends based on demographics and professions.
Understand geographic distribution of transactions.
Enable stakeholders to make informed business decisions through interactive visualizations.

📈 Dashboard Features
1️⃣ Revenue Performance Analysis
Monitor total transaction revenue using KPI cards.
Analyze revenue trends over time.
Identify peak revenue periods.

2️⃣ Customer Spending Analysis
Analyze spending patterns based on:
Gender
Profession
Customer Category
City

3️⃣ Transaction Trend Analysis
Track transaction volume trends.
Compare revenue growth across different periods.
Monitor financial performance indicators.

4️⃣ Geographic Analysis
Visualize city-wise transaction distribution.
Identify regions generating maximum revenue.
Explore geographic spending behavior.

5️⃣ Category-Wise Revenue Analysis
Compare spending across categories.
Identify top-performing revenue segments.
Understand customer purchasing preferences.

6️⃣ Customer Segmentation
Segment customers based on demographics.
Analyze contribution of different professions.
Identify high-value customer groups.

7️⃣ Interactive Filters & Slicers
Users can dynamically explore data using:
Gender Filter
